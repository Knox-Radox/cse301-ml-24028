## Prerequisites

* Python 3.12
* `uv`
* Jupyter Notebook or JupyterLab

---

## Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/cse301-ml-24028.git
cd cse301-ml-24028
```

Install the project dependencies:

```bash
uv sync
```

Launch Jupyter:

```bash
uv run jupyter lab
```

or

```bash
uv run jupyter notebook
```

Open any notebook in the repository and execute the cells sequentially.
